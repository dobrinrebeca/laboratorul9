Use Case: Partajarea materialelor cu echipa

1. Descriere

Acest caz de utilizare permite managerului să partajeze materiale cu echipa pentru a colabora eficient.

2. Actori

  • Manager
  • Echipa de vânzări

3. Pre-condiții

  • Managerul este autentificat pe platformă.

4. Post-condiții

  • Materialul partajat este vizibil pentru membrii echipei.

5. Flux principal

  1. Managerul navighează la secțiunea „Materiale”.
  2. Selectează materialele pe care dorește să le partajeze.
  3. Alege echipa sau membrii cu care dorește să le partajeze.
  4. Apasă pe „Partajează”.

6. Flux alternativ

  • 2a. Managerul nu găsește materialul necesar → folosește funcția de căutare.

Use Case: Generarea raportului de activitate

1. Descriere

Acest caz de utilizare permite managerului să genereze un raport despre activitatea utilizatorilor asupra materialelor.

2. Actori

  • Manager

3. Pre-condiții

  • Managerul are drepturi de vizualizare asupra materialelor.

4. Post-condiții

  • Raportul este descărcat în format PDF.

5. Flux principal

  1. Managerul accesează secțiunea „Rapoarte”.
  2. Selectează perioada de timp și materialele pentru raport.
  3. Apasă pe „Generează Raport”.
  4. Vizualizează raportul și îl descarcă.

6. Flux alternativ

  • 3a. Managerul introduce o perioadă nevalidă → este notificat și trebuie să corecteze selecția.
